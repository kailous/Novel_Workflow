# 📚 小说创作工作区 (Novel Writing Workspace)

欢迎进入您的小说创作环境！本框架旨在帮助您系统化地管理小说的设定与进程，确保长篇连载时逻辑自洽、灵感不断。

## 📁 目录结构说明

[**➡️ 快速跳转：小说章节目录 (Table of Contents)**](小说目录.md)

| 目录/文件 | 用途说明 |
| :--- | :--- |
| **`01_世界观设定/`** | 存放世界背景、基础法则、力量体系（战力等级）、地理势力等底层逻辑。 |
| **`02_角色设定/`** | 存放主角群、反派、重要配角的详细人设档案档案（性格、动机、能力、弧光）。 |
| **`03_道具设定/`** | 存放核心金手指、关键任务物品、武器草药等设定。 |
| **`04_时间线/`** | 记录故事发生的历史大事件，以及主线剧情的详细时间表，防止时间错乱。 |
| **`05_剧情大纲/`** | 包含整体的三幕结构大纲、分卷大纲，把握故事的整体走向和节奏。 |
| **`06_AI助手/`** | 提供预设的 Prompt (提示词) 和工作流指引，用于随时呼叫 AI 检查逻辑冲突、提供灵感或辅助描述。 |
| **`07_插图/`** | 存放与小说相关的插图、概念图等视觉素材。 |
| **`07_插图/画师/`** | 存放根据文字描述生成SD/MJ绘画提示词的专职AI。 |
| **`07_插图/画师/美术总监.md`** | 美术总监的AI配置。 |
| **`07_插图/画师/美术风格指导_井上雄彦.md`** | 统领画面笔触与硬派张力的风格指导AI配置。 |
| **`07_插图/画师/人物角色设计师.md`** | 人物角色设计师的AI配置。 |
| **`07_插图/画师/场景设计师.md`** | 场景设计师的AI配置。 |
| **`07_插图/画师/服装道具设计师.md`** | 服装道具设计师的AI配置。 |
| **`07_插图/(生成的图片库)...`** | 存放AI生成的图片。 |
| **`08_正文/`** | 存放小说的正文内容，按卷或章节组织。 |

## 🚀 推荐的起步流程
1. **第零步**：点击 [小说章节目录](小说目录.md) 查看已创作的章节。
2. **第一步**：打开 `05_剧情大纲/小说大纲.md`，写下你的**一句话简介**和核心爽点。
2. **第二步**：转到 `01_世界观设定/世界观核心.md` 建立基础的战力体系或背景环境。
3. **第三步**：在 `02_角色设定/主角及重要角色.md` 中完善你的主角，确保他有明确的动机。
4. **第四步**：随时参考 `06_AI助手/AI写作工作流.md` 中的提示词，让我（AI）在您卡文或构思时提供帮助。
