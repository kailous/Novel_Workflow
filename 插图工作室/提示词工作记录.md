# 📝 提示词工作记录与生成规则 (针对 Nano Banana Pro 2)

## 1. 目标模型特性分析
**模型名称**：“Nano Banana Pro 2” 对**结构化提示词**的理解极佳，强排版、重风格。

## 2. 提示词 (Prompt) 结构规范 (10行模板)
所有指派给美术团队的绘画要求，最终必须输出为**严格换行**的 10 行（或附加负面词11行）英文关键词：

```
[主体 + 类型 + 动作],
[艺术风格],
[关键形态细节],
[构图方式],
[光线类型],
[色彩方案],
[材质质感],
[背景描述],
[质量标签],
[比例],
[负面约束，如: no text, no anime]
```

## 3. 具体Tag编写规则
1. **纯英文标签输出**：各画师输出时，每行末尾保留逗号，且强制换行。
2. **“凡人流”动漫基调约束**：
   * **[艺术风格]**：`art by Takehiko Inoue, Japanese anime style, high quality manga illustration`
   * **[色彩方案]**：`cool color palette, cyan and dark green tones`
   * **[材质质感]**：`manga crosshatching, cel shading, detailed ink lines`
   * **[规避词汇]** 放最后：`no 3D, no realistic, no bright colors, no glowing effects, no handsome flawless face`

## 4. 模块化整合流程
1. **美术总监** 定下基调和构图（前缀与光影）。
2. **人物画师** 提供主体面部与体态词（主体）。
3. **服装道具画师** 提供服饰材质和特写词（服化道）。
4. **场景画师** 提供背景氛围词（环境）。
5. 最终由整合流程将这些词块拼合，生成完整可用的 Prompt。
